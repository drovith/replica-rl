Rule 01

Dated, not dripped

Drops go live at exactly 20:00 GMT on the published date. No early access, no secret links, no “friends and family.” Everyone hits the same door at the same second.

Rule 02

Open ’til empty

A drop stays live until the final unit is gone — could be nine minutes, could be nine days. We don’t set timers. The run sells out when it sells out.

Rule 03

Sold out is final

Once a piece is gone it moves to the archive below and never returns. The list is the only way to be early. There is no waitlist that turns into a restock.
